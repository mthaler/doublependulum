# doublependulum
The equation of motion and the phase diagram of a pendulum

Positions:

$$
x_1 = l_1\dot sin(\theta_1)
y_1 = -l_1\dot cos(\theta_1)
x_2 =   l_1\dot sin(\theta_1) +  l_2\dot sin(\theta_2)
y_2 = -l_1\dot cos(\theta_1) -l_2\dot cos(\theta_2)
$$
